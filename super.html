Linux Mint에서 쓰는, 유용한 멀티미디어 프로그램들 (Apps for Multi-Media)

밖에 벚꽃이 아주 활짝 피었습니다.
꽃피는 봄날입니다.
이런 화창한 봄날에 멀티미디어를 이용해서 분위기를 좀더 내 볼 수 있겠죠?


오늘은 Linux Mint를 가지고 멀티미디어를 편리하게 이용할 수 있게 해주는
여러가지 프로그램들을 소개해 드리겠습니다.
패키지,프로그램,소프트웨어,어플리케이션,어플,앱.......다 같은 말입니다.
음악도 듣고,영화도 보고,
라디오도 다양한 쟝르별로 마음껏 들을 수 있구요
youtube를 볼때도 한가지 주제를 주욱 보기 쉽게 편리하게 이용할수도 있구요
바탕화면도 시간마다 자동으로 다운받아서 ,때되면 자동으로 바꾸구요

 아이패드나 ,아이폰이면 파일 변환도 해서 옮겨야 하구요
영화 한편에 30분이면 파일 변환도 오케이!! avi -> mov파일일때 이렇더군요 
mp4는? 안해봤습니다.
이런 기능들을 수행 해줄 프로그램들을 알아 보겠습니다.

 프로그램들을 설치하기 전에 가장 먼저 해야 할 일은,
우선,ubuntu-restricted-extras 를 설치 하는 겁니다.
저는 ,Mint나, Ubuntu등을 설치 할때,가장 먼저 설치해 주는 것 중에 하나가 ubuntu-restricted-extras 입니다.
설치 안하신 분들은 우선 ubuntu-restricted-extras 부터 설치하시고 시작해주세요

 apt install ubuntu-restricted-extras ( sudo apt-get install ubuntu-restricted-extras )

 먼저 음악부터 시작하겠습니다.
터미널에서 음악듣기는 moc를 이용하세요,바로 이전의 포스트에서 다뤘었답니다.
http://bagjunggyu.blogspot.kr/2013/04/moc-music-on-console.html

1)Banshee 
iTunes와 아주 닮은 미디어 플레이어입니다.
동영상 재생도 가능 하지만,권장하지는 않습니다.
음악을 듣기 위해서 이용하시면 참 좋은 어플입니다.
인터넷 아카이브 기능을 활용 하시면 아주 좋은 여러가지 동영상 컨텐츠를 무료로 보실 수 있습니다.
음악을 추가 하실때는 음악에서 마우스 오른쪽을 클릭하시면 폴더를 통째로 인식하거나,
파일을 추가 할수 있는 설정이 나옵니다.
파일 브라우저로부터 직접 드레그 앤 드롭은 되지 않습니다.
민트에는 기본플레이어 입니다. 우분투등 다른 os에서 이용 하시려면

sudo apt-get install banshee
해서 설치하시면 됩니다.



위의 그림에서는 모두반복기능과 ,랜덤플레이를 설정해놓았습니다.
앨범 표지를 표시 하거나,
이퀄라이저를 설정하거나,
컨텍스트 영역 - 곡에대한 설명 ,혹은 그와 비슷한 성향의 가수나 곡을 별도로 표시해 주는 기능 입니다.
우리 가요도 지원합니다.
보기(V)에서 원하시는 기능을 설정 하시면 됩니다.

실행 하시면 패널에 실행 아이콘이 나타납니다.
그래서,밴시를 꺼도 배경에서 플레이가 되게 되어 있습니다.
완전히 끌때는 패널의 트레이 아이콘에서 끄면  됩니다.
알림 표시는 재생되는 곡의 정보를(곡명,가수)표시해 주는 기능입니다.
저는 꺼 놓습니다. 아래 그림처럼 알림표시를 해제 하시면 됩니다.

배경에서 들리게 하지 않고, 밴시를 끄면 같이 꺼지도록 하시려면
편집 - 기본 설정 - 확장명에서 '알림 영역 아이콘 '을 해제 하시면 됩니다.

 라디오에 한국 라디오 방송국 채널들을 추가하실 수 있습니다.
물론 주소를 입력해서 하셔도 됩니다.그것과 더불어

 아래의 한국 우분투 포럼 사이트에 가셔서, banshee.db.tar.gz 파일을 다운 받으세요
http://ubuntu.or.kr/viewtopic.php?f=6&t=17546

 1) 다운받은 압축파일을 풀면 banshee.db 파일이 나올겁니다.
2) banshee.db 파일을 복사해서 /home/사용자/.config/banshee-1 에 붙여 넣으셔서 
    원래 있던 파일과 바꾸는 겁니다.

 .config 는 숨김 폴더 입니다.
/home/사용자 폴더에서 Ctrl + H 누르시면 보입니다.


 2)Clementine
클레멘타인은 음악전용 플레이어 입니다.
동영상 재생은 지원되지 않습니다.하지만,라디오 기능이 아주 좋습니다.
사실 라디오 때문에 클레멘타인을 이용합니다.
밴시에서는  파일 불러들이기를 음악에서 파일,폴더추가를 해야만 하지만
클레멘타인에서는 드레그 앤 드롭이 됩니다.라디오는 Internet에 있습니다.
설치는 

apt install clementine ( sudo apt-get install clementine )



 밴시와 마찬가지로 패널에서 트레이 아이콘으로 지원합니다.
꺼도 배경에서 음악이 나오구요,완전히 끄려면 패널에서 종료하시면 됩니다.
클레멘 타인이 꺼질때 같이 꺼지게 하시려면,
도구(T) - Preferences - Behavior 에서 Show tray icon을 해제 하시면 됩니다.


3)Radiotray
라디오트레이는 라디오 전용 플레이어 입니다.
클레멘타인보다 지원되는 채널이 조금 적습니다.하지만,제가 좋아하는 채널이 이곳에 있어서
가끔씩 듣습니다.
설치는,

apt install radiotray (sudo apt-get install radiotray)


4)SMPlayer
ogg 파일은 기본 동영상 플레이어가 제일 좋더군요,
하지만,다른 형식의 동영상 재생은 이게 제일 좋더군요
저는 최신버전인 0.8.4를 이용중입니다.

기본 버전 설치 : apt install smplayer 

smplayer 0.8.4 버전 설치

sudo add-apt-repository ppa:rvm/smplayer
sudo apt-get update
sudo apt-get install smplayer 



재생목록은 플레이어 안이나 밖으로 옮길 수 있게 되어 있습니다.

한글 자막을 설정 하시려면 
옵션 - 환경설정에 가세요

자막으로 가셔서 
기본 자막 인코딩을 한국어 만자집합(CP949) 을 하시구요
Try to autodetect for this language (이 언어를 자동으로 인식하기)에서 Korean(ko)로 설정 하시면 됩니다.

자동으로 자막을 불러들이지 않거나,자막파일을 바꾸시려면
자막 - 불러오기에서, 자막파일이 있는 폴더에 가서 불러오시면 됩니다.
자막 불러오기는 동영상이 재생될때만 가능합니다.


5)Minitube
youtube를 보는 아주 좋은 방법중에 하나가 이렇게 TV의 한 채널처럼 보는 겁니다.
요새 나오는 음식기행 프로중에 테이스티로드를 좋아하는데요,애석하게 방송을 보지는 못합니다.
다행히 youtube에 짤막한 편집 영상이 나와서 그걸 보고 있는 데요
그렇듯이 한 주제를 찾아서 목록으로 이어보기를 가능하게 해준답니다.

최신 버전인 Minitube 2.0 버전을 설치하세요

sudo add-apt-repository ppa:nilarimogard/webupd8
sudo apt-get update
sudo apt-get install minitube



6)Variety
버라이어티는 배경화면 자동 재생기능과 함께,
설정된 시간에 서버에서 배경화면을 자동으로 다운받는 기능까지 있는 바탕화면 꾸미기 도구 입니다.
실행하면, 패널의 알림 영역에 아이콘이 나타나며 그곳에서 설정을 하거나,종료 하게 됩니다.
Preferences에서 원하시는 대로 설정을 변경하세요

기본설정은 그림처럼되어 있습니다.
Change wallpaper every 10 Minutes (10분 마다 배경화면을 바꿉니다)
Images - 배경화면 이미지를 다운 받을 서버 ,모두 체크 하시면 모든 서버에서 다 다운 받습니다.
저는 제 폴더에 있는 것만 보려고 다른 서버나 위치는 해제해 놓은 것입니다.
Download a wallpaper every 10 minutes to Downloaded 
(Downloaded 폴더에 10분마다 배경화면을 다운 받아 놓습니다.)
Limit download folder to 500 MB by deleting oldest files when necessary 
(다운로드는 500MB 까지만 ,필요하면 가장 오래된 파일부터 지웁니다.)
설치는

sudo add-apt-repository ppa:peterlevi/ppa
sudo apt-get update 
sudo apt-get install variety


7)Format Junkie
아주 다양한 기능을 가진 포맷 변환프로그램입니다.
음악파일 변환,동영상 파일 변환,이미지파일 변환,iso 이미지 만들기,등
일상 생활에서 아주 유용하게 사용될 프로그램입니다.
지원되는 파일 형식은 그림으로 보실까요?

음악 파일

동영상 파일

이미지 파일

iso 만들기

동영상에 자막 입히기

꽤 ,쓸만하죠?
동영상 파일 변환은 avi파일 영화 2.4GB를, mov파일로 변환시킬때 (4.9GB 생성) 30분 걸렸습니다.
설치는


sudo add-apt-repository ppa:format-junkie-team/release
sudo apt-get update
sudo apt-get install formatjunkie
입니다.

Arch Linux 설치과정 정리 - Part II


Arch Linux 설치과정 정리 - Part I 에 이어서, 아치 리눅스 설치과정 정리 
이제 두 번째로 본격적으로 우리가 흔히 사용할 수 있는 컴퓨터로 구성하는 과정입니다.
Desktop Environment를 설치해줄건데요
여러가지 Desktop 환경들이 있죠

 Gnome : http://www.gnome.org/
KDE : http://www.kde.org/
Xfce : http://www.xfce.org/?lang=ko
LXDE : http://www.lxde.org/
Cinnamon : http://cinnamon.linuxmint.com/
Mate : http://mate-desktop.org/
등등이요
Ubuntu 의 Unity 와 elementary os 의 elementary는 아직 별도의 데스크탑 환경으로는 제공되지 않고 있습니다.
다만, elementary 는 이번에 Google Summer of Code에서 Debian 용으로 Porting될 예정입니다.
이런 여러가지 그래픽컬 사용자 환경을 이용하려면 우선 X-server를 설치해야 합니다.

 Arch Linux 설치하고 부팅하면, 이런 Grub 화면을 만나게됩니다.

 인터넷 연결을 먼저 확인 하겠습니다.
인터넷 연결을 확인하시구요, 안되어 있으면 연결합니다.
ping -c 3 www.google.com
하시구요
ip addr 또는 ip link 
하시면 네트워크연결에 필요한 정보가 있습니다.
제 경우는 유선기기 enp4s0 입니다. 연결명령은
systemctl start dhcpcd@enp4s0
입니다.

5)x-server 설치
xorg-xinit는 X의 윈도우 시스템 시작도구 이구요 
xterm은 X윈도우의 터미널이구요
xorg-twm 은 X11의 윈도우 매니저 (창관리자) 이구요
xorg-xclock는 X윈도우의 시계입니다.
pacman -S xorg-server xorg-server-utils xorg-xinit xorg-twm xorg-xclock xterm

제 Dell Mini의 그래픽 카드가 Intel 입니다.
pacman -S xf86-video-intel

 제 경우의 그래픽카드에서 3D효과를 이용하려면 mesa를 설치해야합니다.
pacman -S mesa

랩탑의 터치패드를 이용하기 위해서 설치합니다.
pacman -S xf86-input-synaptics

 이제 데스크탑 환경을 설치하니까요, 멋진 폰트를 설치해주면 좋죠
ttf-dejavu를 추천해 드립니다.이후에 DE에 따라서 바뀌게 되지만, 제 Dell Mini에는 이 폰트가
참 잘어울리더군요
pacman -S ttf-dejavu

이제 x 윈도우를 실행해 보겠습니다.
startx
짜잔 이것이 X 윈도우 입니다.

 이제 X 윈도우를 꺼볼까요?
pkill x

인터넷 연결도 아직은 위에서 하시던 대로 수동으로 연결해주세요
정말 단촐하게 딱! 시나몬만 설치되어 있습니다.
gedit 같은 문서도구도, 터미널도 없습니다.
이제 여러가지 패키지들과 설정들을 더해서 좀 더 편리하게 만들어 가겠습니다.

가장 필요한 패키지들 부터 설치해 보려고 합니다.
터미널은 tilda를 쓰려고 합니다. 편집기는 gedit , 압축관리 프로그램은 file-roller
웹브라우저는 chromium 입니다. 기본 터미널인 gnome-terminal은 설치하면 작동을 안하네요
xterm으로 실행해 주세요

sudo pacman -S tilda gedit chromium

tilda의 설정 과정입니다.
Menu의 System-settings -- Custom Shortcuts 에서 Name 과 Command 둘다 tilda로 하시구요

원하시는 단축키를 설정해주세요, 저는 Alt + w 로 했습니다.

이제 Alt + w 를 눌러서 tilda를 실행해주시고 , tilda에 마우스 오른쪽을 클릭하셔서 설정합니다.
저는 이렇게 설정했습니다. F1를 누르면 창을 최소화 합니다.
tilda의 설정에 관해서는 이글을 참고해보세요
http://bagjunggyu.blogspot.kr/2013/05/tilda.html?q=tilda

11) chromium 에 pepper flash plugin 설치
yaourt 는 sudo나 관리자 권한으로 실행하지 않으며, 비밀번호가 필요할땐 yaourt가 물어봅니다.
yaourt -S chromium-pepper-flash


수동으로 설정하기는 아래링크에서 libpepflashplayer.so 파일을 다운받으셔서
https://dl.dropboxusercontent.com/u/54450962/libpepflashplayer.so

/usr/lib/chromium 폴더로 복사하신후에
sudo cp libpepflashplayer.so /usr/lib/chromium

 이렇게 하신후에 chromium 부라우저를 다시 시작해주세요
gedit /etc/chromium/default
# Options to pass to chromium-browser
CHROMIUM_FLAGS="--ppapi-flash-path=/usr/lib/chromium/libpepflashplayer.so"

네트워크 매니저 )
유선이나, 무선 네트워크를 찾아서 연결시켜주는 프로그램 입니다.
가장 편한 방법이죠^^ 설치 부터 해주시구요
sudo pacman -S networkmanager network-manager-applet
sudo systemctl --type=service 를 통해서 실행중인 네트워크 프로그램이 있나 확인합니다.

 실행중인 네트워크 매니저가 있으면 위에서 나온 결과를 바탕으로
sudo systemctl stop 네트워크 매니저 이름 하셔서 중지시킵니다.
또는, 재부팅 하셔서 네트워크를 실행하지 않는 상태로 다시 부팅하신후에 
네트워크 매니저를 실행해주세요 
그럼 네트워크 매니저를 실행해보겠습니다.

 sudo systemctl NetworkManager

이렇게, 네트워크 연결이 가능한 목록이 뜹니다. 그중에 해당하는 네트워크에 연결하시면 됩니다.

 13) 부팅하면서 gdm으로 시작하고 , 네트워크 연결되게 하기
sudo systemctl enable gdm.service
sudo systemctl enable NetworkManager

이제 재부팅 하시면 gdm 으로 시작하고,  네트워크에 연결됩니다.

무선 네트워크 실행 하는 방법 2) 
ipaddr 에 나오는 정보를 바탕으로 입력해주세요 제경우는 wlp3s0 입니다.
dialog 를 설치해야 한다고 하면 sudo pacman -S dialog 하시면 설치됩니다.
sudo wifi-menu wlp3s0

 무선 네트워크 실행하는 방법 3)
확인하셔서 아래의 폴더가 없으면 이렇게 폴더를 만들어 주시구요
sudo mkdir -p /etc/systemd/system/gdm.service.d

만든 폴더안에 이렇게 설정을 만들어줍니다.
sudo gedit /etc/systemd/system/gdm.service.d/after_networkmanager.conf
[Unit]
Wants=NetworkManager.service
After=NetworkManager.service

이러면 gdm 에서 부터 네트워크 매니저를 실행합니다.
